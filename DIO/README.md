## 🤖 COMANDOS

- [git branch -m master main] ➡️ Trocar nomeclatura da branch de master para main
- [git init] ➡️ Iniciar repositorio local
- [git status] ➡️ Verificar commits na branch
- [git remote add origin "URL DO REPOSITORIO"] ➡️ Vincular repositorio local ao remoto
- [git add "Nome do arquivo/pacote"] ➡️ Adicionar arquivo/pacote a estante
- [git reset "Nome do arquivo/pacote"] ➡️ Remove arquivo/pasta da estante
- [git restore "Nome do arquivo/pasta] ➡️ Restaura o arquivo/pasta da estante modified para a versão anterior
- [git commit --amend -m "escrever nova mensagem"] ➡️ Altera a mensagem do ultimo commit
- [git reset --soft "hash"] ➡️ Retornar os arquivos do commit identificados no hash referente e adiciona a estante para commit.
- [git reset "hash" ou git reset --mixed "hash"] ➡️  Retornar os arquivos do commit identificados no hash referente como não identificado(Não está na estante).
- [git reset --hard "hash"] ➡️  Limpa todo o log posterior ao registro hash posto no comando
- [git reflog] ➡️ mostra um histórico detalhando das mudanças feitas na branch
